# MCP Architecture Explanation

Model Context Protocol (MCP) is designed to standardize how AI models interact with external tools.

## Key Components

1. AI Model
2. MCP Client
3. MCP Server
4. Tools / Data Sources

## Workflow

1. Model requests a tool
2. MCP client sends structured context
3. MCP server executes tool
4. Response returned to model

## Benefits

- Security
- Standardization
- Tool interoperability
- Better AI workflows
